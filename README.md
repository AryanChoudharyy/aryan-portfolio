# Aryan Choudhary Portfolio

Premium personal portfolio for Aryan Choudhary, focused on AI engineering, LLM systems, distributed systems, and quantitative products.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lenis

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Local

```bash
npm run dev
```

Open `http://localhost:3000`.
